#Readme

To run the program:

Steps:

    1.)
    use the command --- > "go run filename.go > Output.txt"
        1a)
        Do this to pipe output to a file for an additional function

    2.)
    When prompted for the filename, the file you enter will be opened assuming that there is not a problem with it
        2a)
        You'll want to have the file you wish to open within the directory of the project
        
        2b)
        If there is a problem, then an error will be cast to let the user know
        
        2c)
        If the file does not contain needed indexes for calculations, then an error will be thrown
    
    3.)
    When the file has opened and the program has run, you will be given the appropriate info on screen
